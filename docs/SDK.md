# Stacklane SDK

`@stacklane/sdk` exposes simple v0.4.0 client methods:

- `health()`
- `configStatus()`
- `createCustomer()`
- `listCustomers()`
- `getCustomer()`
- `updateCustomer()`
- `createApiKey()`
- `listApiKeys()`
- `revokeApiKey()`
- `recordUsageEvent()`
- `listUsageEvents()`
- `summarizeUsage()`
- `createAsset()`
- `listAssets()`
- `getAsset()`
- `deleteAsset()`

The SDK targets the `/v1/*` JSON endpoints added in Stacklane v0.4.0.
